# What is gmobin?

***gmobin*** はGMOクリック証券/株価指数バイナリーオプションの非公式APIです。

## Feature
***catsgmo***  GMOクリック証券の下記のサイトのAPI機能を提供します。

1. [株価指数バイナリーオプション デモ取引](https://www.click-sec.com/corp/guide/ixop/demo/)
